# Visual Source Asset Status

This project now has a visual-upgrade intake layer in `visual_upgrade_manifest.json`.

Imported CC0 human upgrades:

- `player_human` -> Poly Pizza / Quaternius Adventurer
- `sister_anwen_human` -> Poly Pizza / Quaternius Animated Woman
- `mira_human` -> Poly Pizza / Quaternius Woman Casual
- `rook_human` -> Poly Pizza / Quaternius Hooded Adventurer
- `villager_human` -> Poly Pizza / Quaternius Animated Human

These assets are legal, lightweight, browser-friendly upgrades over the old OBJ stand-ins. They are still stylized low-poly models, not Witcher-grade scanned/hand-authored humans.

Biggest remaining visual gaps:

- Higher-fidelity human heads, hair, hands, clothing layers, and facial expression rigs.
- Unified animation retargeting for idle, walk, attack, hit, death, and dialogue gestures.
- Hero-specific equipment attachments and clothing variants so named NPCs do not read like generic pack characters.
- Higher-fidelity ghoul/bog monster models with attack/death animations.
- Terrain materials with authored albedo/normal/roughness texture sets for road, mud, grass, stone, and wood.

Preferred future source assets:

- `.glb` or `.gltf`
- Rigged humanoid skeleton
- Idle, walk, attack or gesture animations where possible
- Visible face, hair, hands, clothing, and grounded feet
- CC0, CC-BY with attribution, purchased, or otherwise licensed for redistribution
- Low or mid poly, texture-atlas friendly, suitable for browser export
